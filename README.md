# Keep The Right — sito promozionale

Landing page ufficiale di **Keep The Right**, il quiz di cultura generale con
sfide online 1 vs 1 ("Dimostra che lo sai!").

🌐 **Live**: https://vage88sg1.github.io/keep-the-right-site/

- Statico, zero build: `index.html` + `assets/`
- Hosting: GitHub Pages (branch `main`, root)
- Trailer: https://youtu.be/YPbTId7j_9s
- Privacy policy del gioco: https://keep-the-right.web.app/privacy

## Sviluppo locale

```bash
python3 -m http.server 8787
# → http://localhost:8787
```

screenshot brandizzati). Il badge Google Play è il badge ufficiale di Google;
